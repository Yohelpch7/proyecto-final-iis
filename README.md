* Descripción del proyecto

Este proyecto implementa un sistema escolar básico utilizando el lenguaje de programación C.
El sistema permite administrar estudiantes, cursos, inscripciones, calificaciones y reportes mediante estructuras, arreglos dinámicos y archivos binarios.

Este proyecto cumple con los requisitos solicitados para el trabajo final de la asignatura.

* Funcionalidades implementadas

El sistema cuenta con las siguientes funciones principales:

Registrar estudiante

Dar de alta curso

Inscribir estudiante en curso

Listar estudiantes por curso

Calcular el promedio de un curso

Generar boleta de un estudiante

Mostrar el curso con más estudiantes aprobados

Pruebas básicas del sistema

Guardado y carga de datos mediante archivos binarios (cursos.dat)

Gestión de memoria dinámica (arreglos dinámicos para alumnos y cursos)

Tecnologías utilizadas:

Lenguaje C

Manejo de archivos binarios (fread, fwrite)

Memoria dinámica (malloc, free)

Estructuras (struct)

Archivos incluidos en este repositorio:

main.c – Código fuente del proyecto

proyecto_final.pdf – Reporte final del proyecto

README.md – Este documento


El programa genera el archivo cursos.dat automáticamente para guardar cursos y estudiantes inscritos.

El programa maneja memoria dinámica para evitar límites rígidos en el almacenamiento de alumnos dentro de cada curso.

El proyecto está diseñado para ser claro, organizado y cumplir exactamente con los requerimientos mínimos, sin excederse innecesariamente.
